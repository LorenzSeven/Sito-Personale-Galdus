Creazione di una macchina virtuale con la Cloud Shell

testo Copiato! 

1) Cercare su google: cloud console google
2) Aprire la cloud shell in alto a destra
3) Dalla cloud shell creare la macchina virtuale con questo codice
gcloud compute instances create "vm-cshell-quatela"  \
--machine-type=f1-micro \
--tags=http-server,https-server \
--image-family=debian-11 \
--image-project=debian-cloud \
--preemptible \
--subnet=default \
--zone europe-west1-b
4) aprire la macchina virtuale cliccando SSH sotto la colonna Connetti
5) Dalla macchina virtuale installare nginx con questo comando: sudo apt install nginx
