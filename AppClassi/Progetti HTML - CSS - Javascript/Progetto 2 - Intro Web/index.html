Che cos'è BootStrap?

Bootstrap è una libreria di componenti front end e un toolkit open source. Viene usato per creare pagine web reattive e mobile-first con HTML, CSS e JavaScript. Bootstrap usa tecnologie SASS, flexbox (sistema di griglie flessibili) e mixin (classi che contengono metodi utilizzabili da altre classi, senza che esista un rapporto genitore-figlio tra queste due classi). Bootstrap è un toolkit estremamente diffuso, creato da programmatori Twitter - prima di essere open-source, era infatti conosciuto come Twitter Blueprint. Dal suo rilascio nel 2011, Bootstrap si è evoluto attraverso numerose versioni, con alcune riscritture importanti - con le versioni 2 e 3.

Che cos'è Jquery?

jQuery è una libreria JavaScript cross-browser rilasciata nel 2006. Veloce, versatile ed estensibile, l'API di jQuery consente di gestire con semplicità delle attività che risulterebbero lunghe o complesse in JavaScript. Infatti jQuery racchiude in metodi richiamabili con una sola riga di codice delle attività che in JavaScript avrebbero bisogno invece una lunga programmazione. Inoltre jQuery semplifica eventi come le chiamate AJAX e la manipolazione del DOM che in JavaScript sono di gestione complessa.
