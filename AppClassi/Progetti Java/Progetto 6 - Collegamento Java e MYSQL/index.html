Passaggi per importare il progetto Java in Eclipse

1) scaricare il file zip
2) estrarre la cartella del file zip
3) su Eclipse cliccare su File in alto a sinistra
4) cliccare su Open Projects from File System... 
5) sulla finestra che appare cliccare su Directory...
6) cercare la cartella estratta in precedenza
7) cliccare su Finish
8) ora il progetto Java e' importato in Eclipse

In MYSQL: 
1) creare il database studentiDB
2) usare il database studentiDB
3) crreare la tabella studenti

Script in MYSQL:
create database studentiDB;
use studentiDB;
create table studenti(
id int not null auto_increment primary key,
name varchar(50),
studentid varchar(50),
cource varchar(50)
);

Parti importanti in: jdbc:mysql://localhost:3306/studentiDB","root","Password!"
- localhost: e' l'indirizzo IP
- 3306: e' il numero della porta d'ingresso
- studentiDB: e' il database di riferimento
- root: e' l'username dell'utente
- Password!: e' la password dell'utente

Query: select
insert = con1.prepareStatement("SELECT * FROM studenti");
- studenti: e' la tabella che sta dentro il database studentiDB.

Query: insert
insert = con1.prepareStatement("insert into studenti(name,studentid,cource)values(?,?,?)");
- studenti: e' la tabella che sta dentro il database studentiDB
- name, studentid, cource: sono le colonne della tabella studenti. Id non c'e' perche' autoIncrementante.

Query: select
insert = con1.prepareStatement("delete from studenti where id = ?");
- studenti: e' la tabella che sta dentro il database studentiDB. Condizione where per selezionare la riga da cancellare.

Query: select
insert = con1.prepareStatement("update studenti set name= ?,studentid= ?,cource= ? where id= ?");
- studenti: e' la tabella che sta dentro il database studentiDB
- name, studentid, cource: sono le colonne della tabella studenti. Condizione where per selezionare la riga da cambiare.

Per eseguire il file Java
1) cliccare sulla freccia del progetto Java
2) cliccare su Cource/src
3) cliccare su cource
4) cliccare 2 volte sul file Java
5) cliccare su Run in alto a sinistra per eseguire il file Java
