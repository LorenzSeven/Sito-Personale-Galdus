Esercizio 1 - helloName
Given a string name, e.g. "Bob", return a greeting of the form "Hello Bob!".

}

Esercizio 2 - makeAbba
Given two strings, a and b, return the result of putting them together in the order abba, e.g. "Hi" and "Bye" 
returns "HiByeByeHi".

public String makeAbba(String a, String b) {
  return a + b + b + a;
}

Esercizio 3 - makeTags
The web is built with HTML strings like "< i >Yay" which draws Yay as italic text. In this example, 
the "i" tag makes < i > and  which surround the word "Yay". Given tag and word strings, 
create the HTML string with tags around the word, e.g. "< i >Yay".

public String makeTags(String tag, String word) {
  return "<" + tag + ">" + word + "";
}


Esercizio 4 - makeOutWord
Given an "out" string length 4, such as "<<>>", and a word, 
return a new string where the word is in the middle of the out string, e.g. "< < word > >". 
Note: use str.substring(i, j) to extract the String starting at index i and going up to but not including index j.

public String makeOutWord(String out, String word) {
  return out.substring(0, 2) + word + out.substring(2, 4);
}

Esercizio 6 - firstTwo
Given a string, return the string made of its first two chars, so the String "Hello" yields "He". 
If the string is shorter than length 2, return whatever there is, so "X" yields "X", 
and the empty string "" yields the empty string "". Note that str.length() returns the length of a string.

public String firstTwo(String str) 
{
  String stringa = "";
  if (str.length() >= 2)
  {
     stringa = str.substring(0, 2);
  }
  else if (str.length() == 1)
  {
    stringa = str.substring(0, 1);
  }
  else if (str.length() == 0)
  {
    stringa = "";
  }
  return stringa;
}

{

Esercizio 20 - hasBad

      inizioBad = true;

    if (str.substring(1, 4).equals("bad"))

      inizioBad = true;
    }    
  }
  
  return inizioBad;
}


Esercizio 21 - atFirst
Given a string, return a string length 2 made of its first 2 chars. If the string length is less than 2, 
use '@' for the missing chars.

{
    String stringa = "";
    if (str.length() >= 2 )
    {
      stringa = str.substring(0, 2);
    }
    else if (str.length() == 1 )
    {
      stringa = str.substring(0, 1) + "@";
    }
    else if (str.length() == 0 )
    {
      stringa = "@@";
    }

}


Esercizio 22 - lastChars
Given 2 strings, a and b, return a new string made of the first char of a and the last char of b, 
so "yo" and "java" yields "ya". If either string is length 0, use '@' for its missing char.

public String lastChars(String a, String b) 
{
  String stringa = "";
  if (a.length() == 0 && b.length() == 0)
  {
    stringa = "@@";
  }
  else if (a.length() >= 1 && b.length() == 0)
  {
    stringa = a.substring(0, 1) + "@";
  }
  else if (a.length() == 0 && b.length() >= 1)
  {
    stringa = "@" + b.substring(b.length() - 1, b.length() );
  }
  else if (a.length() >=1 && b.length() >= 1)
  {
    stringa = a.substring(0, 1) + b.substring(b.length() - 1, b.length() );
  }
  return stringa;
}

Esercizio 23 - conCat
Given two strings, append them together (known as "concatenation") and return the result. However, 
if the concatenation creates a double-char, then omit one of the chars, so "abc" and "cat" yields "abcat".

public String conCat(String a, String b) 
{
  String stringa = "";
  if (a.length() == 0)
  {
    stringa = b;
  }
  if (b.length() == 0)
  {
    stringa = a;
  }
  if (a.length() == 0 && b.length() == 0)
  {
    stringa = "";
  }
  else if (a.length() > 0 && b.length() > 0)
  {
    if (a.charAt(a.length()-1) == b.charAt(0))
    {
      stringa = a.substring(0, a.length()) + b.substring(1,b.length() ) ;
    }
    else
    {
      stringa = a + b;
    }     
  }

  return stringa;
}

Esercizio 27 - minCat
Given two strings, append them together (known as "concatenation") and return the result. 
However, if the strings are different lengths, 
omit chars from the longer string so it is the same length as the shorter string. 
So "Hello" and "Hi" yield "loHi". The strings may be any length.

public String minCat(String a, String b) 
{
  String stringa = "";
  if (a.length() == b.length())
  {
    stringa = a + b;
  }
  else if(a.length() > b.length())
  {
    stringa = a.substring(a.length() -  b.length() , a.length() ) + b;
  }
  else if(a.length() < b.length())
  {
    stringa = a + b.substring(b.length() -  a.length() , b.length() );
  }
  return stringa;
}

Esercizio 30 - deFront
Given a string, return a version without the first 2 chars. 
Except keep the first char if it is 'a' and keep the second char if it is 'b'. 
The string may be any length. Harder than it looks.

public String deFront(String str) 
{    
  String stringa = "";
  if (str.charAt(0) == 'a' && str.charAt(1) != 'b' )
  {
    stringa = 'a' + str.substring(2, str.length());
  }
  else if (str.charAt(0) != 'a' && str.charAt(1) == 'b' )
  {
    stringa = 'b' + str.substring(2, str.length());
  }
  else if (str.charAt(0) == 'a' && str.charAt(1) == 'b' )
  {
    stringa = "ab" + str.substring(2, str.length());
  }
  else
  {
    stringa = str.substring(2, str.length());
  }
  return stringa;
}


Esercizio 31 - startWord
Given a string and a second "word" string, 
we'll say that the word matches the string if it appears at the front of the string, 
except its first char does not need to match exactly. 
On a match, return the front of the string, or otherwise return the empty string. 
So, so with the string "hippo" the word "hi" returns "hi" and "xip" returns "hip". 
The word will be at least length 1.

public String startWord(String str, String word) 
{
  String stringa = "";
  if (str.length() >= 2 )
  {
    String strNo1 = str.substring(1, word.length());
    String wordNo1 = word.substring(1, word.length());
  
    if (strNo1.equals(wordNo1))
    {
      stringa = str.substring(0, word.length());
    }
  }
  else if (str.length() == 1 && word.length() == 1)
  {
      stringa = str;
  }
  return stringa;
}

Esercizio 32 - withoutX
Given a string, if the first or last chars are 'x', return the string without those 'x' chars, 
and otherwise return the string unchanged.

public String withoutX(String str) 
{
  String stringa = "";
  if (str.length() >= 2 )
  {
    char carattere1 = str.charAt(0);
    char carattereN = str.charAt(str.length() - 1);
  
    if (carattere1 == 'x' && carattereN == 'x' )
    {
      stringa = str.substring(1, str.length() - 1);
    }
    else if (carattere1 == 'x' && carattereN != 'x' )
    {
      stringa = str.substring(1, str.length());
    }
    else if (carattere1 != 'x' && carattereN == 'x' )
    {
      stringa = str.substring(0, str.length() -1);
    }
    else
    {
      stringa = str;
    }    
  }

  return stringa;
}


Esercizio 33 - withoutX2
Given a string, if one or both of the first 2 chars is 'x', return the string without those 'x' chars, 
and otherwise return the string unchanged. This is a little harder than it looks.

public String withoutX2(String str) 
{
  String stringa = "";
  if (str.length() >= 2)
  {
    char Char1 = str.charAt(0);
    char Char2 = str.charAt(1);    
    
    if (Char1 == 'x' && Char2 == 'x')
    {
      stringa = str.substring(2, str.length());
    }
    else if (Char1 != 'x' && Char2 == 'x')
    {
      stringa = Char1 + str.substring(2, str.length());
    }
    else  if (Char1 == 'x' && Char2 != 'x')
    {
      stringa = str.substring(1, str.length());
    }
    else
    {
      stringa = str;
    }    
  }

  return stringa;
}
